# txt-crawler-bookbao
a small node crawler in bookbao, it helps you to download some type of books in site bookbao  

to get start with :   
`$ npm install`  
`$ node crawler book-type`  

if you just want to read snippets in each book, try  
`$ node crawler book-type your-filter`  
  
# example
`$ node crawler 诛仙 鬼厉`
